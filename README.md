# JavaScript Exercises & Webcam Project

This repository contains the exercises completed during a JavaScript course, along with a final project developed at the end of the course.

The goal of this repository is to document the learning process and demonstrate the practical use of JavaScript concepts through small exercises and a functional webcam application.

---

## 📚 JavaScript Course Exercises

Inside this repository you will find multiple small exercises created while learning JavaScript.  
These exercises were designed to practice and better understand fundamental JavaScript concepts such as:

- Variables
- Functions
- DOM manipulation
- Events
- Basic logic and control structures

The repository also includes examples using different browser APIs such as Geolocation, Battery Status, and machine learning experiments with ml5.js.

Each exercise focuses on a specific topic covered during the course.

---

## 📸 Webcam Project

The main project of this repository is a **Webcam Application** developed using JavaScript and the **Web Camera API**.

This application allows users to access their webcam, capture photos, preview them, and download them individually or as a ZIP file.

---

## ✨ Features

- 📷 Capture photos using the webcam  
- 👀 Preview captured photos in a popup viewer  
- 💾 Download a single photo  
- 📦 Download multiple photos as a ZIP file  
- 🖱 Hover controls with icons for quick actions  
- 🔒 Photo capture disabled when the webcam is not active  
- 📸 Photo limit to manage captured images

---

## 🛠 Technologies Used

- **JavaScript**
- **HTML**
- **CSS**
- **Web Camera API**
- **JSZip** (for downloading multiple images as a ZIP file)
- **Font Awesome** (for UI icons)

## 🚀 How to Run the Project

1. Clone this repository
https://github.com/osmanturkarslan/Javascript_Emvs2.git

2. Open the project folder

3. Run `index.html` in your browser

⚠️ Note:  
Your browser will ask for **camera permission** in order to use the webcam.
